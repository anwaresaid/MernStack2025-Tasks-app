# Task Manager Application

This is a simple Task Manager application that allows users to register, log in, view their tasks, update them, and delete them. Admins can manage all tasks and users.

## Features

### User Features:

- Register and log in
- View, update, and delete their own tasks
- Users can only access their own tasks

### Admin Features:

- Admin can log in with predefined credentials
- View, update, and delete tasks of all users
- Admin has full control over the task list
